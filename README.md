# Health Check API - Node.js + TypeScript

## Descripción
API backend simple creada para practicar y refrescar conceptos de Node.js, Express y TypeScript, sin frontend ni base de datos.

## Tecnologías usadas
- Node.js
- Express
- TypeScript
- Morgan
- Middleware personalizado
- Arquitectura por capas (routes, controllers, middleware)

## Endpoints disponibles
- GET /health
- GET /ping
- GET /server-info
- GET /memory-usage
- GET /cpu-usage
- GET /db-status (simulado)

## Objetivo del proyecto
Practicar:
- Estructura backend profesional
- Middlewares
- Manejo de errores
- Tipado con TypeScript
- Logging

## Cómo ejecutar el proyecto
npm install  
npm run dev
